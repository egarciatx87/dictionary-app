# Pocket Dictionary App
A simple, fast, and responsive dictionary application built to help users explore the English language. This project focuses on fetching real-world data from a third-party API and displaying it dynamically.

# Features
Search Functionality: Instantly look up any word in the English language.

Audio Playback: Click a button to hear the correct pronunciation of the word.

Detailed Information: Displays the word's phonetic spelling, definition, and a list of synonyms.

Dynamic UI: The interface updates instantly without needing to refresh the page.

# Built With
HTML: Structure and content.

CSS: Styling and layout.

JavaScript: Logic, event handling, and API integration.

API: Free Dictionary API

# What I Learned
Building this app helped me bridge the gap between static HTML and dynamic web applications. Key takeaways include:

Asynchronous JavaScript: Using async and await to fetch data from a server without freezing the browser.

JSON Navigation: Learning how to "dig" through complex, nested data structures like data[0].meanings[0].definitions[0].

DOM Manipulation: Using JavaScript to target specific HTML ids and update their content based on user input.

Error Handling: Implementing try...catch blocks to handle cases where a word might not exist in the dictionary.

# How to Use
Enter a word into the search box.

Press the Search button.

View the definition and synonyms.

Click Play Audio to hear the pronunciation.
